## **Creating Voting Option Using MapiMessage**

Microsoft Outlook allows users to create a poll when composing a new message. It allows them to include voting options such as Yes, No, Maybe, etc. Aspose.Email allows the same while creating a new Outlook message. The [FollowUpOptions](https://reference.aspose.com/email/net/aspose.email.mapi/followupoptions/) class provides the [VotingButtons](https://reference.aspose.com/email/net/aspose.email.mapi/followupoptions/votingbuttons/) property that can be used to set or get the value of voting options. This article provides a detailed example of creating a [MapiMessage](https://reference.aspose.com/email/net/aspose.email.mapi/mapimessage/)Â with voting options for creating a poll.
