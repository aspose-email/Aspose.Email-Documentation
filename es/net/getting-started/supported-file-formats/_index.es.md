Aspose.Email para .NET proporciona un conjunto integral de funcionalidades diseñadas para el manejo eficiente de mensajes de correo electrónico y datos asociados. Un aspecto fundamental de la biblioteca radica en su sólido soporte para una diversa gama de formatos de archivo, permitiendo a los usuarios procesar sin problemas datos relacionados con el correo electrónico dentro de sus aplicaciones. A continuación se muestra un resumen de todos los formatos de archivo soportados: 

|**Formato**|**Descripción**|**Cargar**|**Guardar**|
| :- | :- | :- | :- |
|[MSG](https://docs.fileformat.com/email/msg/)|Este formato es utilizado por Microsoft Outlook y Exchange para almacenar mensajes de correo electrónico, contactos, citas u otras tareas. Aspose.Email soporta la lectura, escritura y modificación de archivos MSG.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[PST](https://docs.fileformat.com/email/pst/)|Representa los Archivos de Almacenamiento Personal de Outlook que almacenan correos electrónicos, elementos de calendario, notas, contactos y varios otros formatos de archivo. Se usa para archivar datos de correo electrónico sin conexión. Aspose.Email permite trabajar con archivos PST, incluyendo lectura, escritura y manipulación de datos.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[OST](https://docs.fileformat.com/email/ost/)|Representa los datos del buzón del usuario en modo sin conexión en la máquina local al registrarse con el Servidor Exchange usando Microsoft Outlook. Aspose.Email permite trabajar con archivos OST, incluyendo lectura y manipulación de datos.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[OFT](https://docs.fileformat.com/email/oft/)|Archivos de plantilla de mensajes que se crean utilizando Microsoft Outlook. Aspose.Email permite trabajar con archivos OFT, incluyendo lectura y escritura.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[OLM](https://docs.fileformat.com/email/olm/)|Este formato es utilizado por Microsoft Outlook para Mac para almacenar mensajes de correo electrónico, contactos, diarios, datos de calendario y otros datos de aplicaciones de usuario. Aspose.Email soporta la extracción, manipulación y conversión de los datos contenidos en estos archivos.|{{< emoticons/tick >}}| |
|[EML](https://docs.fileformat.com/email/eml/)|Representa mensajes de correo electrónico guardados usando Outlook y otras aplicaciones relevantes. Aspose.Email proporciona soporte para la lectura y escritura de archivos EML, así como para convertirlos a otros formatos.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[EMLX](https://docs.fileformat.com/email/emlx/)|Es un formato utilizado por Apple Mail para almacenar un solo mensaje de correo electrónico. Aspose.Email permite la lectura y escritura de archivos EMLX.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[MBOX](https://docs.fileformat.com/email/mbox/)|Representa un contenedor para la colección de mensajes de correo electrónico. Aspose.Email proporciona soporte para la lectura y escritura de archivos MBOX.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[ICS](https://docs.fileformat.com/email/ics/)|El formato iCalendar se utiliza para intercambiar y desplegar eventos de calendario y programación. Aspose.Email proporciona soporte para la lectura y escritura de archivos ICS, permitiendo la integración de datos de calendario en aplicaciones.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[VCF](https://docs.fileformat.com/email/vcf/)|El Formato de Tarjeta Virtual es un formato de archivo digital para almacenar información de contacto. Aspose.Email permite a los usuarios leer, escribir y manipular datos de contacto.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[HTML](https://docs.fileformat.com/web/html/)|HTML (Lenguaje de Marcado de Hipertexto) es la extensión para páginas web creadas para su visualización en navegadores. Aspose.Email equipa a los desarrolladores con la capacidad de manejar archivos HTML, facilitando la extracción y uso de contenido de estos archivos dentro de sus aplicaciones.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
|[MHTML](https://docs.fileformat.com/web/mhtml/)|Formato de archivo de archivo de página web que guarda el código HTML y los recursos asociados en un solo archivo. Aspose.Email soporta la lectura de archivos MHTML, permitiendo la extracción de su contenido para su uso en aplicaciones.|{{< emoticons/tick >}}|{{< emoticons/tick >}}|
